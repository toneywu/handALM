键盘操作

全局

  • Tab - 选中到甘特图
  • Alt + 上/下/左/右 滚动甘特图
  • Ctrl + 回车 创建新任务
  • Ctrl + Z - 撤销
  • Ctrl + R - 重做
  • Esc - 关闭弹出窗口

表头

  • 左/右 - 左右导航，滚动日期范围
  • Home/End - 跳转到最早/最后一个单元
  • 下 - 导航到任务中
  • 空格/回车 - 点击表头

任务行

  • 上/下 - 在上下任务之间切换
  • PageDown/PageUp - 跳转到第一个/最后一个任务
  • 空格 - 选择任务
  • Ctrl + 回车 - 创建新任务
  • Delete - 删除当前任务
  • 回车 - 弹出任务编辑界面
  • Ctrl + 左/右 - 展开/收缩任务树层次
